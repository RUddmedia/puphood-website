# Puphood Website

Static marketing + legal site for the Puphood iOS app, hosted on **GitHub Pages**.

## Pages
- `index.html` — landing page (hero, features, contact)
- `privacy.html` — Privacy Policy (linked in App Store Connect)
- `terms.html` — Terms of Service / EULA (subscription terms)
- `styles.css` — shared styles (brand palette sampled from the app icon)
- `assets/logo.png` — app icon
- `CNAME` — custom domain (`puphood.app`) for GitHub Pages

## Brand palette (from the app icon)
- Amber (primary): `#F2991B`
- Deep amber: `#D47C28`
- Golden highlight: `#FED97F`
- Warm brown (text): `#5B4233`
- Cream (background): `#FFF8EE`

## Editing later
1. Edit the `.html` / `.css` files in this folder.
2. Commit and push to the `main` branch.
3. GitHub Pages redeploys automatically within a minute or two.
